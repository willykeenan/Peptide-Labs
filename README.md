# Ascension Labs — demo storefront (RUO)

This is a demo Next.js storefront you can use to gather feedback.

## Run locally
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Notes
- This demo positions products as **Research Use Only (RUO)** and **Not for human consumption**.
- Checkout is wired to a server API route that returns a placeholder URL. Swap in Stripe/other payments when ready.
